# hello_ep

개발환경

- Vue.js

필요 요소

- nodejs https://nodejs.org/ko/
- Vue cli https://cli.vuejs.org/guide/installation.html

## node modules 설치

```
npm install
```

### 로컬 서버 시작

```
npm run serve
```

### Firebase 배포

테스트 서버로 Firebase 사용하여 ( https://console.firebase.google.com/ ) 로 배포했습니다.

### 배포 주소

https://hello-ep-test.web.app/

### 배포 방법

- 빌드 : npm run build
- 배포 : firebase deploy
